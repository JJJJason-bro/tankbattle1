# 🎮 坦克大战 (Tank Battle)

一个使用纯 HTML5 Canvas 制作的经典坦克大战网页游戏，无需任何依赖，打开即玩！

## 🎯 游戏特色

- 🕹️ **经典玩法** — 致敬FC红白机经典《坦克大战》(Battle City)
- 🎨 **精美画面** — Canvas 绘制的坦克、地图、爆炸特效
- 🔊 **音效系统** — Web Audio API 驱动的实时音效
- 🧠 **智能AI** — 4种敌人类型：普通、快速、重型、智能追踪
- 💎 **道具系统** — 加速⚡、速射🔥、护盾🛡️、加命❤️
- 🗺️ **随机地图** — 每关自动生成砖墙、铁墙、水域、树林
- 📡 **小地图** — 右上角实时显示敌我位置
- 📱 **移动端支持** — 触屏虚拟摇杆控制
- ⏸️ **暂停功能** — 按 P 键暂停/继续

## 🎮 操作说明

| 按键 | 功能 |
|------|------|
| `↑ ↓ ← →` / `W A S D` | 移动坦克 |
| `空格` / `J` | 发射子弹 |
| `P` | 暂停/继续 |
| `R` | 重新开始 |

## 🚀 快速开始

1. 克隆仓库：
```bash
git clone https://github.com/YOUR_USERNAME/tank-battle.git
```

2. 直接用浏览器打开 `index.html` 即可开始游戏！

或者直接访问 GitHub Pages 在线游玩。

## 🎯 游戏规则

- 消灭关卡中所有敌方坦克即可通关
- 保护你的基地（底部中央的金色鹰徽标志）
- 基地被摧毁则游戏结束
- 收集掉落的道具增强能力
- 每关敌人数量和难度递增

## 🛠️ 技术栈

- HTML5 Canvas
- CSS3 (动画、渐变、响应式)
- Vanilla JavaScript (ES6+)
- Web Audio API
